A — One-liner per source

Each source lands as a single line. Colour-coded: red = risk, blue = info, green = clean. Fast to scan. Pending sources show ⏳ until they resolve. Dense but readable.

FINDINGS — ethiks360.com

B — Grouped by category

Findings arrive and are grouped into categories (Email, Breach, TLS, etc.) with key-value pairs under each. More structured. Feels like a proper intelligence report being typed out live.

FINDINGS — ethiks360.com
gaps found so far:

C — Gap-first with context

Only significant findings shown. Each one is a mini-finding block: gap title + one-line explanation. Pending sources shown as a single summary line. Right pane tells a story, not just data.
